ปัญหาที่ SME เจอเวลาคิดจะใช้ AI Chatbot ครั้งแรก คือ **ไม่รู้ว่าจะเริ่มยังไง** เปิดเว็บ vendor ขึ้นมาก็เจอ pricing tier 5–6 อัน ฟีเจอร์เต็มหน้าจอ ดูแล้วงง เลยปิดหน้าจอกลับไปขายแบบเดิม

KORP AI เปิดให้ลองสดผ่าน 3 template ที่ทำมาตรงกับธุรกิจไทยจริง ๆ — เข้าไปคลิกได้เลย ไม่ต้องสมัคร ไม่ต้องกรอกอีเมล ไม่ต้องคุยกับ sales ก่อน:

## 3 template ลองสดได้

### 1. ร้านเสื้อผ้าออนไลน์ — `/demo/fashion`
- catalog สินค้า + ราคา + size guide
- order tracking + stock flow
- AI ตอบคำถามไซส์ สี ส่งไปต่างจังหวัดได้ไหม
- sales orders dashboard ดู conversion rate รายวัน

เหมาะกับร้านเสื้อผ้า กระเป๋า รองเท้า เครื่องประดับ — ตั้งแต่ตู้เสื้อผ้าเจ้าของคนเดียวจนถึง brand ที่มี SKU 500+

### 2. ร้านอาหาร / คาเฟ่ — `/demo/restaurant`
- menu + รูปภาพ + ราคา (อัพเดทเองได้)
- table booking ผ่าน Line OA
- AI ตอบลูกค้า: เปิดกี่โมง? มีที่จอดรถไหม? เมนูแพ้ถั่วได้หรือเปล่า?
- รายงานยอดขายประจำวันส่งเข้า Line ตอนปิดร้าน

เหมาะกับร้านอาหาร ร้านกาแฟ shop ที่อยากให้ลูกค้าจองโต๊ะได้ตลอด 24 ชม. โดยไม่ต้องมีคนรับสาย

### 3. ธุรกิจทั่วไป (B2B / บริการ) — `/demo/business`
- lead capture form + auto-response
- ใบเสนอราคาอัตโนมัติ (กรอกข้อมูล → AI generate quote PDF)
- KPI dashboard
- handoff ส่งต่อ sales ทันทีเมื่อลูกค้าจริงจัง

เหมาะกับ agency, ที่ปรึกษา, ธุรกิจ SaaS, บริการที่ต้องคุยกับลูกค้านานก่อนปิดดีล

## ทำไม template ดีกว่าเริ่มจากศูนย์

วิธีเดิม: brief → spec → design → build → test → launch = 4–8 สัปดาห์ + งบประมาณ 50,000+

วิธี template: เลือก template ที่ใกล้ธุรกิจตัวเอง → ปรับแค่ branding (สี logo) + ใส่ข้อมูลจริง (menu/สินค้า/ราคา) → เปิดใช้ได้ใน 1–2 สัปดาห์ + งบประมาณเริ่มต้นต่ำกว่ามาก

ส่วนของ AI Sales Agent ที่ตอบลูกค้า — ไม่ต้องเขียน prompt เอง template มาพร้อม system prompt ที่ทดสอบกับลูกค้าจริงแล้ว 6+ เดือน รู้ว่าจะตอบยังไงให้ลูกค้าไม่หงุดหงิด รู้จังหวะส่งต่อทีมขายเมื่อเจอ buying signal

## ขั้นตอนหลังลอง template

1. **ลองสด** — เข้า /demo เลือก template → กด chat ลองคุยกับ AI → ดูว่าตอบสไตล์ไหน กดปุ่มไหนแล้วเป็นยังไง
2. **ทักมาคุยโจทย์** — Line / Facebook → บอกว่าธุรกิจอะไร ลูกค้าเดือนละกี่คน ใช้งบเท่าไหร่
3. **เสนอ scope + ราคา** — ภายใน 24 ชม. ส่งใบเสนอราคา + timeline
4. **เริ่มงาน** — kick-off → ส่ง draft ให้ดูทุก 1 สัปดาห์ → ทีมลูกค้าเทส → adjust → launch

ทดลอง 14 วันแรกหลัง launch ถ้าไม่พอใจคืนเงินเต็ม ยกเลิกรายเดือนได้ไม่มีค่าปรับ

## คำถามที่เจอบ่อยตอนลอง template

**Q: ปรับสี/โลโก้ให้ตรงแบรนด์ตัวเองได้ไหม?**
A: ได้ทั้งหมด — เปลี่ยนสี / font / โลโก้ / ทุก copy ภาษาไทย

**Q: AI ตอบเฉพาะภาษาไทยไหม?**
A: ตอบได้ทั้งไทย + อังกฤษ (สลับอัตโนมัติตามภาษาลูกค้า)

**Q: รองรับ Line OA + FB Messenger พร้อมกันไหม?**
A: รองรับ — ตั้งแต่แพ็กเกจ Growth ขึ้นไป AI ตัวเดียวคุมทุกช่องทาง บทสนทนาต่อเนื่องข้ามช่องทางได้

**Q: ลูกค้าถามนอกเหนือจาก template เดิม ทำยังไง?**
A: เพิ่ม knowledge base ได้ — อัปโหลด PDF / Google Doc / Sheet แล้ว AI จะตอบจากเอกสารนั้นเพิ่ม
