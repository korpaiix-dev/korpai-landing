Why this set

Thailand's OIC (สำนักงาน คปภ.) requires insurance sale/recommendation to come from a licensed agent — AI chatbots can collect leads and answer public-information FAQs only. PDPA s.26 requires explicit consent for sensitive personal data (health, biometric). This set is the minimum production package KORP AI ships when onboarding a new insurance agent/broker chatbot.
