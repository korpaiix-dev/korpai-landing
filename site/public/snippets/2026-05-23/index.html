Snippets — 23 May 2026

Open-source helpers for building a 2026-modern AI chatbot for a Thai outbound travel agency. All snippets MIT-licensed. Companion to the blog post AI Chatbot ตัวแทนท่องเที่ยว/Travel Agency SME ไทย 2026.

  • visa-rule-guardrail.json — system-prompt addendum + refuse-patterns to keep the bot from guaranteeing visa approval
  • multi-currency-quote-lock.ts — 24-hour locked quote with rate snapshot in THB / USD / EUR / JPY / CNY / KRW
  • group-booking-pdpa-consent.py — per-passenger consent batch flow (PDPA s.19)
  • tat-license-widget.html — drop-in TAT license verification widget with link to ททท. registry
  • itinerary-builder-prompt.md — LLM system prompt for 3-tier itinerary draft + visa lookup
  • mfa-travel-advisory-monitor.sh — poll MFA / State Dept advisory feeds and webhook to n8n / Line OA
  • refund-routing.ts — IATA / non-IATA / hotel-direct / OTA / agency-package refund decision tree
Use at your own risk. None of these are a substitute for a TAT-licensed travel agency, an embassy, or proper legal advice. Always verify visa rules with the destination embassy and PDPA compliance with a qualified DPO.
