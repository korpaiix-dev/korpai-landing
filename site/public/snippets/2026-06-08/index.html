โค้ดประกอบบทความ AI Chatbot ความปลอดภัย (Prompt Injection + ข้อมูลรั่ว) สำหรับ SME ไทย 2026 — ป้องกัน 8 ชั้น — แต่ละไฟล์คือ 1 ชั้นของ defense-in-depth, รันได้จริงด้วย Node.js / Python.

  • prompt-injection-detector.js — ชั้น 3: ตรวจจับ injection/jailbreak (ไทย+อังกฤษ) regex-first, 0 token
  • input-sanitizer.js — ชั้น 1+2: แยกข้อมูลผู้ใช้ออกจากคำสั่ง + ตัด zero-width/control char
  • pii-output-redactor.js — ชั้น 4: redact เลขบัตร ปชช./เบอร์/บัตรเครดิต/อีเมล ก่อนส่งออก (Luhn + checksum)
  • cross-user-firewall.py — ชั้น 5: กันข้อมูลข้ามลูกค้า (IDOR) — ผูก query กับ identity เสมอ
  • tool-call-allowlist.js — ชั้น 6: least-privilege ให้บอตเรียกได้เฉพาะ action ที่อนุญาต
  • rate-limit-guard.js — ชั้น 7: token-bucket กันยิงรัว/ดูดข้อมูล/ค่า token บานปลาย
  • injection-test-suite.md — red-team checklist 6 หมวด รันก่อน launch + regression test

อ้างอิง: OWASP Top 10 for LLM Applications 2025 (LLM01 Prompt Injection, LLM02 Sensitive Information Disclosure, LLM06 Excessive Agency).
